# TP/0: Time Definition Protocol

This repository contains the source for `draft-wang-tp-definition-00`, a conceptual framework for representing and manipulating time in AI systems.

## Abstract

The Time Protocol (TP) family defines a four-layer architecture for cognitive time in AI systems: perception, direction, copy, and emergence. This document (TP/0) provides the foundational definitions, terminology, and layer relationships.
